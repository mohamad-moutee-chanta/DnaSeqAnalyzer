# DnaSequenceAnalyzer:

A python tool that explains basic molecular biology properties of a given DNA sequence

# What if provides:

- Calculation of GC content 
- Calculation of GC/AT skew 
- Generation of the Complementary Strand
- Generation of the mRNA (Transcription)
- Generation of the protein (Translation)
- Molecular Weight Calculation
- Hydrophobicity analysis using the Kyte-Doolittle scale
